This was a pretty fun assignment. I've never used Node's streams before so I thought that they would be a good way of implementing this server. I think that was a good choice because my server could serve multiple files at one time and the client starts getting the file as I'm reading it, rather than after it is all done.

I also noticed that the caching really made a speed boost. This could be in part that I was installing something and the installer was hoging my disk because normally my servers, even appache, don't take seconds to load.

I enjoyed learning about the Node standard library. It seems very small in comparison to the c++ standard library, but it is well suited for web development. It covers the basics that you encounter while doing web development, and then the NPM has everything else you might need.

The only resource that I needed was the Node documentation. It has all of the standard library in it.

I'm looking forward to working on the Final Project and learning more about databases and server side Node.
